# 🎨 BAND-MAID Creations

This repository collects my **fan-made and AI-generated BAND-MAID visuals**, organized by theme and folder

The repo is published through **GitHub Pages** as an interactive gallery:  
👉 **[View the Gallery](https://drivetimebm.github.io/BAND-MAID_creations/)**

---

## 🖼️ Gallery Features

- 🗂️ **Folder navigation** (two levels deep)  
  Browse by main category and sub-folder.  
- 🖼️ **Automatic thumbnails**  
  All supported image types display in a responsive grid.
- 🔍 **Full-size image viewer**  
  Click a thumbnail to open a large overlay view.
- 💬 **Tooltips**  
  Hovering over any image shows its file name.
- 🪶 **Lightweight & client-side only**  
  Everything runs in the browser—no backend required.

## 🗂️ Repository Structure

| Folder | Description |
|:--|:--|
| **8-bit** | Pixel-art and retro-style BAND-MAID images |
| **gifs** | Animated BAND-MAID GIFs and short loops |
| **Letters** | Alphabet extracted from the logo and AI-generated fan characters (possible font use) |
| **logo** | BAND-MAID logo concepts and design variations |
| **Memes** | Humorous or meme-style BAND-MAID edits |
| **misc** | Miscellaneous creations and images |
| **photos** | Edited and generated BAND-MAID photos by member |
| **ribbon** | Ribbon logo and variations |
